DISPLAY NAME VISIBLE FOR ALL VERIFIED BUSINESSES

(Available in Indonesia, Mexico, and Columbia. This feature will be available globally in 4-8 weeks.)
The recent WhatsApp release mentioned that all existing and new businesses that have completed business verification and display name review will show their WhatsApp Business display name instead of their phone number in the chat list, chat screens, chat groups, and contacts view for any customers that have not added them to their contacts.

Improved value of marketing notifications:

These features will enable faster scaling, enable better campaign insights, and reduce quality disruptions.

Performance metrics, including message sends and reads at the template level and top block reason, were rolled out in WhatsApp manager (for embed users).

Edit live templates : Businesses will be able to edit live templates via WhatsApp manager. Businesses will no longer need to go through the template creation, submission and approval process again if they need to make an edit to their templates.

Increasing messaging limits faster. Subscription to the phone_number_quality_update webhook will notify about the increasing limits.

UPDATES TO CATALOGUE :

WhatsApp has added quantity controls to the list view in Multi-Product Messages, which now allows a customer to add items to their cart without having to click through to the Product Detail Page.

ADDITIONAL PHONE NUMBERS CAN START MESSAGING IMMEDIATELY FOR VERIFIED BUSINESSES :

Businesses that have completed business verification can add additional phone numbers and begin messaging immediately with up to 250 conversations per day on each number. Once the display name review is complete, this limit will automatically be updated to 1,000 conversations. Businesses can now send marketing messages, transactional notifications and run Click to WhatsApp campaigns from the get-go. Business Verification only needs to be completed to scale up messaging limits or to become an Official Business Account.

UPDATES TO CATALOGUE :

Templates once applied should get live immediately. If a duplicate template is submitted, it will be rejected with a notification that indicates the ID of the existing template with the same content. Continue to comply with policies to avoid user reports that may disrupt live templates

ONBOARDING NEW VERTICALS STARTING INCLUDING PHARMACIES & DRUGSTORES AND DIGITAL SUBSCRIPTIONS

Businesses from the following verticals will now have access to the WhatsApp onboarding, with the listed restrictions:
1.

Pharmacies and Drugstores can onboard for all use cases including commerce features for convenience and grocery items, but may not promote or transact in the sale of pharmaceuticals in any messaging or features
2.

Digital Services and Subscriptions can onboard for all use cases except commerce features
3.

Supplements can onboard for all use cases except commerce features
4.

Animals (pet adoption, livestock, raw products, but excluding sale of pets and other non-livestock animals) can onboard for all use cases except commerce features

Additionally Business Messaging policy will solely govern which businesses can onboard and how they use business messaging on WhatsApp. The Commerce policy will determine which businesses can access commerce features like catalog and payments (when available in the future, TBD).
