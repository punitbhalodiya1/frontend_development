Hello Partner

We’re excited to bring you the latest updates and developments we have made on our partner offerings. For our ISV partners, we have expanded our support for a multitude of Meta’s latest features, message types, stability improvements and so on.

Below are a few key enhancements and innovations we recently launched -

1

Partner portal analytics enhancements

On popular demand, we have now added the option in Partner Portal analytics to download CSV. Also apart from existing fields, we have added new columns in the CSV such as the capping amount for the app, Cumulative bill achieved for the month (till capping is achieved), and daily discount provided.

2

Catalog Message Template now supported

Catalog templates are marketing templates that allow you to showcase your product catalog entirely within WhatsApp. You can apply for Catalog templates and send them through the existing Partner endpoints or through the Gupshup self-serve interface

Refer Meta docs

Start implementation

Note : This feature is supported only for WABAs on Cloud API hosted by Meta.

You must have inventory uploaded to Meta in an e-commerce catalog connected to your WhatsApp Business Account. You can link your catalog to your Gupshup WABA by writing to the Gupshup Partner Support team.

3

Catalog Session Message now supported

Similar to Catalog template, catalog session messages allow you to showcase your product catalog entirely within WhatsApp as a free form message. You can send them through the new self serve endpoint here

Refer Meta docs

Start implementation

Note : This feature is supported for WABAs on Cloud API hosted by Meta and On-premise.

You must have inventory uploaded to Meta in an ecommerce catalog connected to your WhatsApp Business Account. You can link your catalog to your Gupshup WABA by writing to Gupshup Partner Support team.

4

Multi Product Message (MPM) Template now supported

Multi product message templates are marketing templates that allow you to showcase up to 30 products from your ecommerce catalog, organized in up to 10 sections, in a single message. You can apply for MPM templates and send them through the existing Partner endpoints or through the Gupshup self serve interface

Refer Meta docs

Start implementation

Note : This feature is supported for WABAs on Cloud API hosted by Meta and On-premise. You must have inventory uploaded to Meta in an ecommerce catalog connected to your WhatsApp Business Account. You can link your catalog to your Gupshup WABA by writing to Gupshup Partner Support team

5

Multi Product (MPM) and Single Product (SPM) Session Message now supported

Similar to MPM template, MPM and SPM session messages allow you to send a free-form messages containing information about a single product or up to 30 products from your product catalog. You can send them through the new self serve endpoint here

Refer Meta docs

Start implementation

Note : This feature is supported for WABAs on Cloud API hosted by Meta and On-premise.

You must have inventory uploaded to Meta in an ecommerce catalog connected to your WhatsApp Business Account. You can link your catalog to your Gupshup WABA by writing to Gupshup Partner Support team.

6

Request address CTA now supported (India and Singapore)

Address messages give your users a simpler way to share the shipping address with the business on WhatsApp. You can send them through the new self-serve endpoint here

Refer Meta docs

Start implementation

Note : This feature is supported for WABAs on Cloud API hosted by Meta and On-premise.

7

Name and Address in Incoming Location Payload Supported

Location request messages through which users can share location were previously supported. In the incoming payload, now we have included 'Name' and 'Address' fields too.

Refer Meta docs

Start implementation

Note : This feature is supported for WABAs on Cloud API hosted by Meta and On-premise.

8

Up to 80 TPS on Cloud API hosted by Meta supported on Self-Serve APIs

Throughput in Gupshup is now aligned with Meta, supporting 80 TPS by default on Self Serve, and 1000 TPS on request to Meta, if the hosting is on Cloud API hosted by Meta.

9

Template Quality Events Added

All self serve apps will receive new type of status update events when template quality changes- 'Template Quality' under System > Template events if they are subscribed to it
