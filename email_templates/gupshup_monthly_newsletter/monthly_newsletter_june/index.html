A lot’s been happening at Gupshup, here’s a wrap-up!

Features

3 new signup tracks added

Updated policy agreement

Before you can move on to step two in the go-live flow, you have to agree to the WhatsApp Business Policy and the WhatsApp Commerce Policy.

Mark Inbound Message as Read

You can now mark inbound messages as read using an API. Find out more.

Free Tier Conversation

WhatsApp will migrate from a notification-based pricing model to a conversation-based (PBC) pricing model. Instead of charging for notifications sent, companies will charge per conversation, which includes all messages delivered in a 24-hour period.

A user guide for the Free Tier conversation type is included. Read it here.

Integrated conversation type for free tier conversations.

You can now determine if a FTC conversation is UIC/BIC using the conversationType parameter in the billing event. UIC/ BIC has been included in the billing-event payload to detect the free tier communications. Read it here

Bug Fixes

Marking inbound messages as read

There were problems with marking new messages as read, but those have now been fixed.

Pagination

The dashboard for more than ten different applications has been updated with a patch for the difficulties that were experienced with pagination.

This allows users to understand the sequence of documentation.

Search bar updation

Issues with the search bar in the template list have been resolved. Now you may search by body content, and it's no longer case sensitive.

Go-live flow

Fixed bug in the go-live flow where WABA is awaiting permission yet the UI states to complete setup.

Upcoming Developments

FB hosted API

Send and receive messages using a cloud-hosted version of the WhatsApp Business Platform. The Cloud API allows you to implement WhatsApp Business APIs without the cost of hosting your own servers and also allows you to more easily scale your business messaging. You can read more about this here

Ability to see messaging limit and tier information for each App

Showcase the existing limit of the app on the dashboard so that the user is not blindsided by the fact that there is a cap placed on the total number of BICs that can be used with the application.

Simplified Template categories

META has now reduced the category section to only 3 - Marketing, Promotional, OTP.
