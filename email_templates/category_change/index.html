How to handle the template category change

Embed Signup WABAs

Created Before April 1, 2023

For approved and rejected templates

Follow the steps here to submit your category change request via the WhatsApp Manager

Created After April 1, 2023

For approved templates

Delete the existing template and submit fresh template with required category.

For rejected templates due to Incorrect category

Follow the steps here to submit your request before May 15.

Traditional Gupshup Flow (OBO)

Created Before April 1, 2023

For approved and rejected templates

Clone the existing template and make minor edits and submit as a fresh template with required category.

(If you wish to delete the template reach out to partner.support@gupshup.io)

Created After April 1, 2023

For approved templates

Delete the existing template and submit fresh template with required category.

For rejected templates due to Incorrect category

Use the Edit Template API for partners to submit with a new category.

If you are having trouble with this, please request to partner.support@gupshup.io for raising an appeal or helping you with the category change requests.

To avoid rejections due to incorrect category, allow category change to Meta while creating template via 'Submit Template API' for partners. Refer docs here

DO NOT clone and submit an existing template as is without deleting the old one, else it can be rejected with error ‘Invalid category’ due to duplication

Always follow WhatsApp Template Guidelines
