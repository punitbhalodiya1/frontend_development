Flujo tradicional de Gupshup (OBO)

Creado antes del 1 de abril de 2023

Para plantillas aprobadas y rechazadas

Clona la plantilla existente, realiza pequeñas modificaciones y envíala como una plantilla nueva con la categoría requerida.

(Si deseas eliminar la plantilla, ponte en contacto con partner.support@gupshup.io)

Creado después del 1 de abril de 2023

Para las plantillas aprobadas

Elimina la plantilla existente y envía una nueva con la categoría requerida.

Para plantillas rechazadas por categoría incorrecta

Utiliza la API de edición de plantillas para partners para enviar una nueva categoría.

Si tienes problemas con esto, puedes dirigirte a partner.support@gupshup.io para elevar una solicitud o para que te ayuden con las solicitudes de cambio de categoría.

Para evitar rechazos debidos a una categoría incorrecta, permite el cambio de categoría a Meta al crear la plantilla a través de la "API de envío de plantillas" para partners. Consulta los documentos aquí

NO clones y envíes una plantilla existente tal cual sin eliminar la antigua, de lo contrario puede ser rechazada con el error "Categoría no válida" debido a la duplicación.
