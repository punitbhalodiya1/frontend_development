Para templates aprovados

Use a API de edição de template para parceiros para enviar com a nova categoria.

Se você tiver dificuldades, peça ajuda com o e-mail partner.support@gupshup.io para abrir um recurso ou ajudar você com as solicitações de mudança de categoria.

Para evitar rejeições por categoria incorreta, permita que a Meta mude categorias ao criar seu template através da API para envio de template para parceiros. Confira a documentação aqui

NÃO duplique e envie um template existente sem excluir o antigo, caso contrário ele pode ser rejeitado com o erro "Categoria inválida" devido à duplicação.
