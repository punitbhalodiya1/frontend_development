Recent Developments on Gupshup

New APIs to use Meta’s Utility Template Library

Meta’s utility library is updated with pre-defined Utility Templates for common use cases like delivery updates, account updates, and reminders. These templates allow for quick creation without the need for approvals, as long as there are no changes in the template. This enhancement streamlines the template creation process for users making them get instant approvals.

Read docs

New API to get WABA info

Get WABA Info is a new API information like accountStatus, dockerStatus, messagingLimit, phone, wabaId, phoneQuality for a given app.

Read docs

New API to get Template Analytics

Partners can use this API to get template analytics for a template after switching on the analytics on Meta for the particular template. This API will give you the button clicks and the exact delivered/read count for a given time range.

Read docs

New API to delete template by template ID & element name

Partners can use this API to delete a template using the templateId and elementName.

Read docs

Low balance alert in the wallet

A low balance alert can now be set to be notified to the main billing email ID, for the desired amount

User Management in Partner Portal

Enforcements from Gupshup

Gupshup is mandating the process of Tech Provider Program to be completed on Meta and Gupshup before 31st October, 2024. Prepare for /sm endpoints End of Life by 31st Aug 2024

As notified earlier, all /sm endpoints starting with 'https://api.gupshup.io/sm' will come to an end of life soon. Deadline for the same has now been extended to 31st August 2024. To ensure future scalability, we need partners to adopt these technical changes in their integrations for supporting upcoming WhatsApp features faster. Please read this article for a seamless migration to /wa endpoints

Prepare for Sunset of Opt-in Opt-out service by Gupshup before 31st Aug 2024

We advise partners to NOT register opt-in opt-outs via any single/bulk upload APIs or get a list of opt-ins through any self-serve or partner APIs. This is irrespective of cloud API or on-premise hosting. Please refer to this article for more details.

Straight from Meta

[COMING SOON] Maximum 2 marketing templates to a user for businesses in India - starting Aug 29

WhatsApp will soon limit sending a maximum of 2 marketing template messages in a 24 hour conversation window without a customer responding. If a customer responds, free-form messages should be used to continue the conversation.

Meta's Authentication Solution Live in India

As of July 1, Meta's authentication solution is live in India! Businesses now can create and use authentication templates to send one-time passwords (OTPs) to users in India (+91 country code). Businesses can configure one-tap, copy-code, and zero-tap buttons on Android for a seamless authentication experience. WhatsApp Authentication rates and Authentication-International rates apply as applicable.

Updates to Marketing and Utility Rates on the WhatsApp Business Platform

Meta has updated rates for marketing and utility conversations to enhance the communication experience between businesses and users. Starting August 1, they have lowered utility rates to encourage more end-to-end post-purchase customer journeys on WhatsApp. Effective October 1, Meta is updating marketing rates in specific markets to better reflect the value delivered by marketing messages. This will lead to an increase in marketing rates in the UAE, Saudi Arabia, and India, and a decrease in rates in the UK. Check the new rate card here

NOTE: Meta has also informed us that, moving forward, rates may be updated more frequently, up to quarterly, with approximately 30 days' notice.
