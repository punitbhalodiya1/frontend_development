Важные изменения в потоке Go Live

Прекращение OBO

Партнеры больше не смогут регистрироваться от чужого имени (OBO) при активации нового номера телефона или его переносе из другого BSP. Встроенная регистрация теперь будет единственным методом. Мы скоро предоставим способ перехода существующих приложений OBO на встроенную регистрацию.

Прекращение On-Premise

Партнеры больше не смогут использовать локальный хостинг при добавлении нового номера телефона или его переносе из другого BSP. Cloud API скоро станет единственным вариантом хостинга. Обратитесь в службу поддержки партнеров, чтобы перенести существующие приложение с локального на облачный хостинг.

Примечание. Обязательно настройте конечные точки отправки сообщений V2 здесь для плавного перехода на Cloud API. В этом выпуске также исправлена ошибка сообщений с эмодзи, связанная с v2.

Изменения в миграции с других BSP на Gupshup

Наряду с изменениями выше, при добавлении нового номера телефона или после его переноса из другого BSP, ОБЯЗАТЕЛЬНО выберите соответствующий вариант использования в начале процесса, чтобы обеспечить правильное выполнение шагов на серверной стороне для каждого случая. Это обеспечит полное получение шаблонов и других данных из FBM при миграции и позволит службе поддержки быстрее и эффективнее оказывать помощь.

Примечание. Для переноса номера телефона OBO из исходного BSP в Gupshup, мы просим вас выполнить миграцию с OBO на встроенную регистрацию на исходном BSP до начала миграции на Gupshup.

Узнайте здесь, как будет работать процесс Go Live.

Комиссия за доставку или прочтение WhatsApp

Gupshup начнет взимать комиссию WhatsApp по статусу доставки или прочтения, в зависимости от того, что произойдет раньше. Обратите внимание, что комиссия Gupshup будет по-прежнему взиматься при отправке сообщения.

Предварительно утвержденные лимиты овердрафта

Мы также вводим предварительно одобренные лимиты овердрафта на основе истории использования и пополнения кошелька за последние 3 месяца.

Gupshup будет назначать предварительно рассчитанный лимит овердрафта в начале каждого месяца, что будет отображаться в интерфейсе кошелька. Пользователь может запросить лимит, равный или меньший рассчитанному, который будет показан как "назначенный" лимит.

Примечание. Если Gupshup назначил лимит вручную, он будет действовать независимо от рассчитанного лимита.

Узнайте здесь как будут работать предварительно утвержденные лимиты овердрафта.

Cloud API теперь доступен в Турции

Meta объявила, что Cloud API теперь доступен в Турции. Это означает, что бизнесы в Турции могут начать работать в WhatsApp, а пользователи с турецкими номерами могут отправлять и получать сообщения через Cloud API. Партнеры, обслуживающие турецкие бизнесы, теперь также могут участвовать в программе Tech Provider

Процесс настройки Tech Provider для партнеров, не входящих в TPP

Портал партнеров позволит партнерам, которые не завершили регистрацию в программе Tech Provider на Gupshup, завершить настройку и привязать новый номер телефона или выполнить миграцию с Joint Solution ID для любого связанного приложения партнера. Узнайте больше в этом руководстве.

Создание демо-приложения для Tech Providers

Tech Provider теперь может пометить приложение как демо для личного использования на партнерском портале Gupshup При запуске он сможет выполнить регистрацию на своем собственном FBM без совместного решения. Узнайте больше в этом руководстве.

Обновления от Meta

Ограничение маркетинговых переписок на одного пользователя

Meta представила новые меры по улучшению взаимодействия с маркетинговыми шаблонами в WhatsApp во всем мире путем ограничения количества сообщений, которые пользователь получает от любого бизнеса в течение определенного периода. Узнайте больше в этом руководстве.

Категоризация шаблонов:

С июня 2024 года Meta будет идентифицировать шаблоны, которые, по их мнению, не соответствуют рекомендациям по классификации шаблонов и уведомлять владельцев WABA о выявленных несоответствиях за месяц до ожидаемых изменений. Этот будет происходить 1-го числа каждого месяца. Узнайте больше в нашем электронном письме «Обновление Meta: ежемесячные изменения в категориях шаблонов».
