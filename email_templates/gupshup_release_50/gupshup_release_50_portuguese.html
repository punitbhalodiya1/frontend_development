1

Apresentando um novo tipo de botão para as mensagens de modelo no WhatsApp: "Copiar código"

Agora você pode usar botões para o usuário copiar um código de cupom nas mensagens de modelo para marketing. Quando o usuário toca ou clica no botão "copiar código", o cupom é copiado para sua área de transferência. Você pode criar este modelo através da API dos parceiros e da interface Gupshup Self-Service.

Confira a documentação para a API Meta

Começar a implementação

Obs.: Somente suportado para a API de nuvem hospedada pela Meta e para as categorias de Marketing. Os códigos têm limite de 15 caracteres máximos.

2

Apresentando um novo tipo de botão para as mensagens de modelo no WhatsApp: "Lista de botões"

As listas de botões, nas mensagens de modelo, oferecem diferentes tipos de botões, como CTAs, respostas rápidas ou copiar código do cupom. Você pode criar este modelo através da API dos parceiros e da interface Gupshup Self-Service.

Confira a documentação para a API Meta

Começar a implementação

Obs.: Suportado tanto para hospedagem local (on premise) e para a API de nuvem hospedada pela Meta. Somente modelos de MARKETING ou UTILIDADE podem ter vários botões de tipos diferentes. Os modelos podem ter até 10 botões, com certos limites para cada tipo e combinação. Os modelos CATÁLOGO e MPM só podem ter um botão.

3

Aumento nas medidas de segurança

Na Gupshup, temos um compromisso em fortalecer a nossa plataforma. Com isto em mente, otimizamos nossas medidas de segurança e solucionamos bugs a fim de garantir uma experiência mais resiliente e flexível para nossos usuários. Também adicionamos algumas funcionalidades de segurança para facilitar a escalabilidade da plataforma.

4

Resolvido: Bug de mídia com legenda

Quando uma mensagem de mídia com documento era enviada juntamente com uma legenda/corpo de texto, uma mensagem duplicada da legenda estava sendo enviada automaticamente. Este erro já foi resolvido. Agora você pode enviar documentos com legenda/texto em uma só mensagem. Confira a documentação aqui.
