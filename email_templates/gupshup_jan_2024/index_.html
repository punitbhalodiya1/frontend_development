Hello Partner

We’re excited to bring you the latest updates and developments we have made on our partner offerings. For our ISV partners, we have expanded our support for many of Meta’s latest features, message types, stability improvements, and so on.

Below are a few key enhancements and innovations we recently launched -

New Features on CAPI

1

Support for Limited Time Offer Templates

Limited-time offer templates allow you to display expiration dates and run countdown timers for offer codes in template messages, making it easy to communicate time-bound offers and drive customer engagement.

Refer: Meta API Docs

Start implementation

Note: Supported only on Cloud API hosted by Meta. Only templates categorized as MARKETING are supported.

2

Carousel Templates

Carousel templates allow you to send a single text message accompanied by up to 10 carousel cards in a horizontally scrollable view.

Refer: Meta API Docs

Start: implementation

Note: Supported only on Cloud API hosted by Meta. Only templates categorized as MARKETING and UTILITY are supported.

New Features on Gupshup hosted Dockers

1

Introducing WhatsApp Business Account Health Page

WABA details such as docker running, ban status, quality rating, tier and so on will now be conveniently available in the Gupshup Self Serve Dashboard. Note: The data will be based on WABA events received, and not fetched in real-time.

2

Template Quality Details

View crucial information such as Template Quality directly in the Template Panel to plan your campaigns better.

3

Visual View of Commissions in Partner Portal

Gain deeper insights and plan to earn higher commissions with the help of visualized data in the Partner Portal.

4

Wallet Nomenclature Changes

Instead of the amount of balance in 'USD', now your Gupshup wallet balance will be visible as 'credits', with 1 USD = 1 credit.

Note: There is no pricing change with this, only nomenclature changes.

5

Additional Billing Email IDs

Partner wallet is now capable of adding up to 3 additional billing email IDs will be provided, apart from the original billing email ID, to receive billing-related communication from Gupshup.

6

Wallet Statement via Email

While downloading wallet statements from the interface will not be possible anymore now, the wallet statement of the selected date range will be sent to the billing email ID/s.

Thank you for being a part of Gupshup, and we hope these updates enhance your experience.
