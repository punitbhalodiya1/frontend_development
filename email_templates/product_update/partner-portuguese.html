Olá Parceiro,

Estamos prontos para as mudanças de categorias e precificação da Meta. Você também? E isso não é tudo — também incluímos algumas funcionalidades interessantes para melhorar sua experiência na Gupshup.

Confira o que foi incluído!

Evento "enviado" e payload de cobrança
Requer alteração técnica antes de 1° de junho de 2023

Mudanças no valor de conversações e objetos de preço para eventos "enviado" e deduções no objeto do evento "cobrança". Leia sobre aqui

Evento para status "modelo de mensagem pausado"

A Gupshup passará a enviar um evento para o status "pausado", junto com a duração da pausa, se a Meta pausar o seu modelo de mensagem. Leia a documentação aqui

Criação de modelos de mensagem para autenticação

Passa a ser possível solicitar modelos para autenticação apenas no formato dado pela Meta, a partir de 29 de maio de 2023. Confira a documentação da API para parceiros para ver como solicitar modelos de autenticação. Você também pode solicitar modelos de autenticação através da interface da Gupshup. Leia sobre aqui

Edição de modelos de mensagem pela interface da Gupshup ou pela API de Parceiros para edição de modelos

Você pode editar todos os campos, exceto Nome, Categoria, Idioma e a caixa "Permitir mudança de categoria" para modelos de mensagem aprovados, rejeitados e pausados. A mudança de categoria é permitida se o modelo tiver sido rejeitado com o motivo "Categoria inválida." Leia a documentação na API de parceiros aqui

Alterações na carteira

Confira este documento para ver as alterações na carteira para as novas categorias e precificação que passam a vigorar em 1° de junho de 2023.

Nova opção para mensagem de solicitação de localização em conversações de serviço

Este tipo de mensagem interativa introduzido recentemente pela Meta ajuda empresas a solicitarem a localização do usuário com um botão "Enviar localização" no qual o usuário pode tocar para compartilhar sua localização. Confira a documentação da API aqui

API de parceiros para uso do Get App
Requer alteração técnica antes de 1° de junho de 2023

Alteração nos objetos de resposta se os dados solicitados forem após 1° de junho; alguns objetos também serão descontinuados se os dados solicitados forem anteriores ao 1° de junho. Leia mais aqui

Alterações em analytics para o Portal de Parceiros

A interface de análise de dados passará a incluir os novos tipos de categorias de conversação. Além disso, os tipos FEP e FTC serão substituídos com as categorias de conversação "Paga" e "Gratuita", com subdivisões disponíveis quando você baixa o relatório. Leia mais aqui
