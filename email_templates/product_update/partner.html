We’re all set for the Meta Category and Pricing Changes - Are you? That’s not all, we have also added some cool helpful features to make your Gupshup experience better.

Have a look at what’s included -

Sent event and billing
payload
Requires technical change before June 01, 2023

Value changes in conversations and pricing objects of sent event and deductions object of billing event. Read here

Paused template status
event

Gupshup will now send you ‘paused’ status event along with duration of pause, if Meta pauses your template. Read docs here

Authentication template creation

Now apply for authentication templates only in the given format by Meta starting 29th May 2023. Refer API doc for partners to refer how to apply for auth templates. You can also submit auth templates through the Gupshup interface. Read here

Edit templates through Gupshup interface or through Partner Edit Template API

Edit all fields except Name, Category, Language and Allow category change checkbox now for approved, rejected and paused templates. Category change allowed if template rejected due to “Invalid Category”.
Read partner API docs here

Wallet changes

Refer this document to see the changes in wallet for the new categories and pricing w.e.f June 01, 2023.

New Location Request message support in service conversations

This recently introduced interactive message type by Meta helps businesses request location of the user in a Send location button that users can tap and share their location. Refer API docs here

Partner API for Get App
Usage
Requires technical change before June 01, 2023

Changed response objects if data requested is after 1st June and some objects will be deprecated if data requested is of before 1st June. Read here

Partner Portal Analytics
changes

The analytics interface will now accommodate the new type of conversation categories. Also, the FEP and FTC types will now be replaced with Paid and Free conversations with breakups available when you download the report. Read more here
