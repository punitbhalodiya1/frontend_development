As part of our continuous efforts to enhance the user experience, we have recently enhanced our user interface to make it more user-friendly, intuitive and accessible. The changes are now live.

For WABAs with multiple phone numbers, you will now get the ability to manage a common pool of templates shared across the phone numbers / Gupshup apps, instead of creating separate templates for each phone number. This means that you can also send templates through the same template ID across the respective Gupshup apps.

Note: For same templates created separately on phone numbers within same WABA till now, those templates will not be auto-merged. You can choose to delete the duplicate ones if you wish to.

Also, along with inbound media url used till now - filemanager.gupshup.io, the inbound media payload will also be coming from domain media-api.gupshup.io. Request you to kindly do the handling.

Please note: we will gradually deprecate filemanager.gupshup.io domain for whichever existing apps are receiving it.

If you have any questions or comments regarding these changes, please contact us at devsupport@gupshup.io
