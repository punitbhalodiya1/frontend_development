User management

Admins

The Admin is able to set up, control and change all settings and processes of a FDX box

Datamanagers

The Datamanager is able to to control all copy/transcode processes of the projects he is assigned to

Cloud Users

The Cloud User is only able to download files from the cloud of projects he is assigned to
